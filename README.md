# Spotify Clone by Yahya 🎵

A **modern web-based Spotify Clone** built using **HTML, CSS, and JavaScript**, designed to replicate the look and feel of the Spotify interface. This project focuses on UI/UX, interactive music controls, and responsive design to provide a near-authentic music streaming experience.

---

## 🔹 Features

- **Play/Pause Music**: Interactive audio playback controls.
- **Next/Previous Tracks**: Navigate through songs seamlessly.
- **Progress Bar**: Displays song progress and allows seeking.
- **Song List**: Browse a curated list of tracks.
- **Responsive UI**: Works on mobile, tablet, and desktop.
- **Modern Design**: Inspired by Spotify’s sleek interface.

## 🛠️ Installation

1. **Clone the repository:**
```bash
git clone https://github.com/gujjardevelopmentspace/spotify-clone-by-yahya.git
````

2. **Navigate to the project folder:**

```bash
cd spotify-clone-by-yahya
```

3. **Open `index.html` in your browser.**

---

## 🧩 Usage

* Click on the play button to start a song.
* Use the next/previous buttons to switch tracks.
* Drag the progress bar to seek within a song.
* Explore the song list and select your favorite tracks.
